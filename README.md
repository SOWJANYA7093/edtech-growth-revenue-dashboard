# EdTech Growth & Revenue Performance Dashboard

This project analyzes an EdTech dataset to understand subscriber growth, revenue trends, and course monetization using Power BI.

## Key Insights
- Subscriber growth peaked around 2015, driven mainly by Web Development courses.
- Paid courses dominate the catalog compared to free courses.
- Web Development contributes the highest revenue among all subjects.
- Average course price and total subscribers are highlighted using KPI cards.

## Tools Used
- Power BI
- Data Modeling
- DAX (basic measures)
- Data Visualization & Dashboard Design

## Dashboard Highlights
- KPI Cards: Total Revenue, Total Subscribers, Average Course Price
- Line Chart: Subscriber growth over years
- Bar Charts: Revenue by subject, Subscribers by course
- Donut Chart: Paid vs Free courses distribution

## Purpose
This dashboard was built as part of my data analytics learning journey to practice real-world dashboard design and storytelling.
